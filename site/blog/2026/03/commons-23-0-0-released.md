<Post>

[Wren Security Commons 23.0.0](https://github.com/WrenSecurity/wrensec-commons/releases/tag/23.0.0)
and [Wren Security Script 5.0.0](https://github.com/WrenSecurity/wrensec-script/releases/tag/5.0.0) have been released.

These are major releases that mark the migration of our foundational libraries to the Jakarta EE
specification. As Commons and Script serve as the base for all other Wren Security projects, this
release unlocks the path for major upgrades across our flagship products like Wren:AM, Wren:ICF,
Wren:IDM, and Wren:IG.

## Changes

List of significant changes:

* Migration to Jakarta EE dependencies
* Migration to Eclipse Angus
* Upgrade Grizzly HTTP framework
* Upgrade Swagger + switch to OpenAPI
* Add JDK 25 support (JDK 17 is now the minimum supported version)
